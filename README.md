# MY BLOG

## How to use

- Clone this repository

```
git clone https://github.com/rintosaputro/my-blog.git
```

- Go to folder repository

```
cd my-blog
```

- Install modules

```
npm install
```

- Run app

```
npm run dev
```

- If the browser does not open automatically then open the browser and go to http://localhost:3000
- Done
